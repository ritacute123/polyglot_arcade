# Polyglot Arcade

Voice-first gamified language proficiency app with a secure Node.js backend.

## Run locally

1. Open a terminal in this folder.
2. Create a `.env` file:

```bash
OPENAI_API_KEY=your_openai_api_key
```

3. Start the app:

```bash
node server.js
```

4. Open:

```text
http://localhost:3000
```

## Important

Do not share `.env`. It contains your private API key.
